**Customer_Behaviour_Predection**
Customer Behavior Prediction project using Python, Pandas, Jupyter Notebook, PostgreSQL, SQL, and Power BI. Features end-to-end data cleaning, database storage, SQL-based analytics, interactive dashboards to generate actionable business insights.

**Business Problem**

Businesses generate large volumes of customer transaction data but often struggle to transform it into actionable insights.

The objective of this project is to:
1. Clean and preprocess customer data
2. Store structured data in PostgreSQL
3. Perform SQL-based business analysis
4. Build an interactive Power BI dashboard for decision-making
